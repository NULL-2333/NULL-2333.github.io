C++中的sort函数的原型

sort(iterator1, iterator2) - 对数组进行递增(non-descending)排序

sort(iterator1, iterator2, comp) - 对数组按照给定的comp方式进行排序

sort函数的第三个参数

第三个参数为cpmparision function object，相当于bool cmp(const Type1 &a, const Type2 &b)，如果第一个变量比第二个小，则应当返回true。一般有三种方式：

standard library comparison function object

custom function object

lambda expression
